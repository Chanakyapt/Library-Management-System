Importance library management system

  • Existing system
    1. Library is managed mannualy.
    2. More time consuming system because of manual work.
    3. The librarian has to perform each task.
    4. Issuing and reciving book is also complex.
    5. Penalty system is very complex.

  • As we see, the existing system following drawbacks which create the need for a new system to:
    1. Consume less time
    2. User-friendly search system
    3. Proper data management
    4. Perfect calculation of penalty with maximum accuracy
    5. Generating monthly/yearly report
    6. maintaining register properly

  • Hence, we use library management system to overcome drawbacks.
